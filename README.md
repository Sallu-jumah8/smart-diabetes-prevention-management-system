# Smart Diabetes Prevention & Management System (SDPMS)

A System Analysis & Design project focused on diabetes prevention and health management through AI-powered features and layered system architecture.

## Project Scope
The system was designed to support:
- AI-based diabetes risk assessment
- Intelligent health chatbot
- Health monitoring dashboard
- Healthcare provider portal
- Notifications and alerts
- Patient-provider communication
